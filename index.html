8086 Microprocessor

This is a Java program to emulate the execution environment of Assembly language instructions in the 8086 Microprocessor. It is multithreaded to handle the asynchronous operations of the Bus Interface Unit and Execution Unit.

Status

Currently all ASM instructions other than JMP type instructions have been added. The input interface isn't created yet, and thus instruction lines cannot be parsed at the moment.
